newsletters ✦

reads your gmail inbox and surfaces daily newsletters as minimal cards. needs a google oauth client id to get started.

  1. go to console.cloud.google.com → create a new project
  2. enable the gmail api (APIs & Services → Library)
  3. go to OAuth consent screen → External → add your google account as test user
  4. create OAuth 2.0 Credentials → Web Application
  5. add http://localhost:8080 to Authorized JavaScript Origins
  6. copy the Client ID and paste below
